# Gevork Akopyan — QA Portfolio

A bilingual personal portfolio built with Next.js, TypeScript, Tailwind CSS and Framer Motion. The site is statically exported and ready for GitHub Pages.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The static site is generated in `out/`.

## GitHub Pages

Pushing to `main` starts the deployment workflow in `.github/workflows/deploy.yml`.

Repository settings must use **GitHub Actions** as the Pages source.
